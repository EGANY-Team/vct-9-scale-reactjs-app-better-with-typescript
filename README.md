## Cài đặt

```bash
# Clone repo
git clone https://github.com/EGANY-Team/vct-9-scale-reactjs-app-better-with-typescript
cd vct-9-scale-reactjs-app-better-with-typescript

# Install packages
npm install

# Start dev server
npm start

# Build
npm run build
```
